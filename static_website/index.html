Motor Control Unit

The motor control unit is a Kinetis K22F 120MHZ ARM Cortex M4 that communicates motor and sensor control commands over USB-CDC using Google Protocol Buffers. The interface clocks at 1.25MBps so that it is possible to get quick closed-loop PID control from collected sensor data.

Vehicle Control Unit

The vehicle control unit is an Android application that speaks to the web via SSH by command line or Google Protocol Buffers. It also provides a webcam, microphone/speaker remote-pilot option.

Power System

The power system is a fast-charging, 24V, 15Wh LiFePO4 battery with a 5V, 2A external power bus in USB form factor. This allows you to attach the Android and various sensors to the frame and draw power from the single source.
